## Getting Started

### Pre Requisites

Make sure you have this installed on your local machine
- nodejs, npm
- mysql

### Executing program

* Prepare the project
  ```
  $ git clone https://github.com/nathanngl/typescript-node-test.git
  $ cd typescript-node-test
  $ npm install
  ```
* Run the app
  ```
  $ npm start
  ```
